# VSCode Units Settings Extension

Provides:
- 🟡 Hover tooltips for `upstream_path` entries
- 🟢 Ctrl+Click / Cmd+Click jump to the corresponding path in `values.yaml`

### Usage

1. `cd tools/vscode-units-extension`
2. `npm install`
3. Open in VSCode: `code .`
4. Press `F5` to open the Extension Development Host

🛠 It reads from:
- `charts/sylva-units/schemas/units-settings.yaml`
- `charts/sylva-units/values.yaml`
